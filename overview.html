solana4j Library Overview

This library provides APIs for reading and writing messages to the Solana blockchain with both the Legacy and V0 Solana Message encodings. This library supports interacting with a subset of the core Solana Programs, including the SystemProgram, TokenProgram and AssociatedTokenProgram. This library can be easily extended to support more Solana Programs.

Features

  • Creating Token, Mint, MultiSig and Nonce Accounts.
  • Sol and Token Transfers.
  • Mint Operations.
  • Nonce Account Operations.
  • Creating, Extending and Retrieving AddressLookupTables.
  • Compute Budget Operations.
